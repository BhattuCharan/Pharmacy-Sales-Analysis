# 🏥 Pharmacy Sales Performance Analysis Dashboard (2022–2024)

## 📌 Project Overview

This project presents a **Power BI analytics dashboard** built using a **synthetic AI-generated pharmacy sales dataset** representing transaction-level activity across the Indian healthcare market from **2022 to 2024**.

The objective of this project is to demonstrate:

* Data preparation and validation
* Business KPI analysis
* Regional and product-level performance evaluation
* Inventory and distribution insights
* Sub-category demand trends
* Executive-level storytelling using Power BI

The final deliverables include:

* Interactive Power BI dashboard
* Structured insights presentation (PPT)
* Analytical summary documentation
* Dataset generation prompt for reproducibility

---

## 📊 Dataset Description

The dataset simulates pharmacy sales behavior across multiple operational dimensions:

* Pharmaceutical brands
* Dosage-form categories and sub-categories
* Discount strategies
* Cities and regions across India
* Units sold and profitability metrics
* Monthly transaction timelines

It contains **2,000 records** and supports analysis of:

* Revenue performance
* Profit contribution
* Inventory demand trends
* Geographic distribution strategy
* Seasonal product usage patterns

⚠️ **Note:**
This dataset is **AI-generated** and intended strictly for **portfolio and analytics practice purposes**.
The original dataset generation prompt has been included in this repository to ensure transparency and reproducibility.
